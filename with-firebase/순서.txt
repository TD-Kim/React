0. firebase 홈페이지 접속해서 계정 생성
1. npm install -g firebase-tools 로 firebase 설치
2. npx firebase login 으로 firebase에 로그인
 2.1 안된다면 노드버전 업그레이드
 2.2 npm에 남아있는 캐시를 우선적으로 삭제해야한다.
     npm cache clean --force 명령어 사용
 2.3 npm install -g n 으로 n 설치(mac, linux 만 가능)
 2.4 윈도우라면 nvm 사용해야함
 2.5 nvm 검색해서 설치
 2.6 nvm install lts 로 노드 최신버전 설치
 2.7 nvm list 로 사용가능한 노드 버전 확인
 2.8 nvm use 로 버전 선택
  2.8.1 nvm use 로 버전 안바뀌면 기존 nodejs 폴더 찾아서 삭제 후 다시 nvm use 사용
        (기존 nodejs 폴더는 보통 C:\\ProgramFiles\\nodejs 폴더로 있음)
